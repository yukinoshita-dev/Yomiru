# Yomiru 手動 QA チェックリスト

## 動作環境
- スマートフォン（iOS Safari / Android Chrome）
- PC ブラウザ（Chrome / Firefox / Safari）
- PWA インストール後

## 1. ライブラリ機能
- [ ] .txt ファイルをドラッグ&ドロップでインポートできる
- [ ] .epub ファイルをインポートできる
- [ ] 青空文庫 .txt ファイルをインポートできる
- [ ] 同じファイルを再インポートしても重複しない
- [ ] サンプル書籍が初回起動時に表示される
- [ ] 書籍カバーが自動生成されて表示される
- [ ] 書籍を削除できる

## 2. リーダー機能
- [ ] 書籍をタップするとリーダーが開く
- [ ] 文節が中央に表示される
- [ ] 自動送りが動作する
- [ ] 左タップで前の文節に戻る
- [ ] 右タップで次の文節に進む
- [ ] 中央タップで再生/停止する
- [ ] 上スワイプで速度が下がる（遅くなる）
- [ ] 下スワイプで速度が上がる（速くなる）
- [ ] 進捗バーが正しく更新される
- [ ] ルビ（ふりがな）が正しく表示される

## 3. キーボード操作（PC）
- [ ] Space で再生/停止
- [ ] ArrowRight で次へ
- [ ] ArrowLeft で前へ
- [ ] ArrowUp で速度を下げる
- [ ] ArrowDown で速度を上げる
- [ ] S キーで睡眠モード切替
- [ ] F キーでフルスクリーン切替
- [ ] Escape でライブラリに戻る

## 4. 設定機能
- [ ] 設定ドロワーが右クリック/長押しで開く
- [ ] 速度スライダーが動作する
- [ ] 文字サイズスライダーが動作する
- [ ] テーマ切替（light / dark / lamp）が動作する
- [ ] 睡眠モードが有効になる
- [ ] 設定ページで全設定が変更できる
- [ ] 設定リセットが動作する
- [ ] データ全削除が動作する

## 5. 睡眠モード
- [ ] 睡眠モード ON で速度が徐々に遅くなる
- [ ] 画面が徐々に暗くなる
- [ ] 指定時間後に自動停止する

## 6. PWA・オフライン
- [ ] ホーム画面へのインストールができる（iOS: 共有→ホームに追加）
- [ ] インストール後にアプリとして起動できる
- [ ] オフライン状態でサンプル書籍が読める
- [ ] オフライン状態でインポート済み書籍が読める
- [ ] 読書進捗がオフライン後も保持される

## 7. アクセシビリティ
- [ ] prefers-reduced-motion 時にフェードアニメーションが無効になる
- [ ] 進捗バーに role=progressbar が付いている（DevTools で確認）
- [ ] スクリーンリーダーで現在の文節が読み上げられる（aria-live）

## 8. パフォーマンス
- [ ] 大きなファイル（100KB超）のインポートが完了する
- [ ] リーダーで長時間再生しても動作が重くならない
- [ ] ページ読み込みが3秒以内
