# ADR-0003: 利用規約・著作権表示の追加

**日付**: 2026-05-25  
**ステータス**: 承認済み  
**担当**: T-033

---

## コンテキスト

Yomiru は利用者が任意のテキストファイル（.txt / EPUB / 青空文庫）をアップロードして読書する PWA。  
そのため、**第三者の著作物を権利者の許可なくアップロードされるリスク**を、サービス運営側として明示的に拒絶しておく必要がある。

特に重要なのは:
- DRM・コピーガード回避テキストの拒絶
- 違法入手テキストの拒絶
- データはローカル完結（サーバー送信なし）である旨の透明性

## 決定

### 必須文言（権利者・利用者向け）

> アップロードできるのは、利用者自身が権利を有するテキスト、著作権の保護期間が満了したテキスト、または権利者から利用許諾を得たテキストに限ります。
> 第三者の著作物を権利者の許可なくアップロードすること、DRM・コピーガードを回避して作成したテキストを利用すること、違法に入手したテキストを利用することは禁止します。
> アップロードされたテキストは利用者のブラウザ内で処理され、当サービスのサーバーには保存されません。

### 表示箇所

| 場所 | 表示内容 | 目的 |
|---|---|---|
| `/terms` ページ（新規） | 利用規約の全文 | 法的根拠の明示 |
| トップページ フッター | `/terms` への小さなリンク | アクセス導線 |
| 初回利用時の同意モーダル | 必須文言の要約＋同意ボタン | 同意取得の証跡（localStorage） |
| ImportDropzone 直下（任意） | 1行リマインダー | アップロード直前の最終警告 |

### 同意の永続化

- `localStorage` キー: `yomiru:terms:acceptedAt` に ISO8601 タイムスタンプを保存
- 値が存在しない場合のみモーダル表示
- 規約を改定したらキー名を `yomiru:terms:acceptedAt:v2` のようにバージョン付けて再同意を求める

## 検討した代替案

1. **モーダルなし・規約ページのみ** → 同意の証跡が残らないため却下
2. **アップロード毎に確認ダイアログ** → UX劣化、初回モーダル＋アップロード画面の注意書きで十分
3. **規約全文をトップに常時表示** → Andon ランディングの世界観を損なうため却下
